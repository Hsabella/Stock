# 名词解释

> 所有项目里出现的专业术语，按使用场景分类，用人话解释

---

## 一、状态机相关

| 术语 | 含义 | 例子 |
|---|---|---|
| **NONE** | 普通候选股，既没买也没特别关注 | 全市场扫描时新发现的票 |
| **WATCHING** | 已加入关注但还没买入 | 你点了"自选"但没下单 |
| **HELD** | 持仓中（你已经买入了） | 雪人集团 600 股，成本 24.818 |
| **EXITED** | 刚卖出，处于"冷静期"不再买回 | 止损后 10 个交易日内不允许 BUY 同一只 |
| **cooldown** | 冷静期天数，避免追涨杀跌反复横跳 | `cooldown_days: 10` |

### 决策动作

| 术语 | 含义 |
|---|---|
| **BUY** | 建议买入（NONE/WATCHING 状态） |
| **ADD** | 建议加仓（HELD 状态，已有票表现强） |
| **HOLD** | 继续持有，啥也别动 |
| **REDUCE** | 建议减仓（不是清仓，比如卖一半锁利润 / 减小风险） |
| **STOP** | 止损（持仓票破位，建议清仓控制亏损） |
| **TAKE** | 止盈（持仓票到达目标位，落袋为安） |
| **CONTINUE_WATCH** | 继续观察（候选票暂时未到买点） |
| **DROP** | 建议从关注池移除（已经走弱，不值得继续盯） |
| **AVOID** | 永远别碰（基本面硬否决，比如连亏 2 年） |
| **NO_ACTION** | 啥也不做 |

## 二、6 个维度

### 1. fundamental_dim（基本面）
**评估什么**：这家公司值不值钱、赚不赚钱

| 术语 | 含义 |
|---|---|
| **PE_TTM** | 市盈率（TTM = 滚动 12 个月）。股价 / 每股收益。**越低越便宜**。亏损股 PE 是负数 |
| **PB** | 市净率。股价 / 每股净资产。常用于银行/资源股 |
| **ROE** | 净资产收益率。1 块钱净资产能赚多少。**越高公司越能赚钱**，>15% 算优秀 |
| **EPS** | 每股收益 |
| **PE 自身分位** | 当前 PE 在该股过去 5 年中排第几位（百分位）。**自身越低越便宜** |
| **PE 行业分位** | 当前 PE 在同行业所有股票中排第几位。**行业横向越低越便宜** |
| **Hard Veto** | 硬否决：触发任一直接 AVOID，不再排名 |
| `veto_loss_2y` | 连续 2 年净利润为负（明显垃圾） |
| `veto_high_goodwill` | 商誉/总资产 > 50%（高商誉炸雷风险） |
| `veto_high_leverage` | 资产负债率 > 80%（非金融业，过度负债） |
| `veto_st` | 名称含 ST（被特殊处理，退市风险） |

### 2. fund_flow_dim（主力资金）
**评估什么**：大资金（机构/北向/游资）在买还是在卖

| 术语 | 含义 |
|---|---|
| **主力净流入** | 大单+超大单的买入金额减卖出金额。**正数=主力在买** |
| **北向资金** | 通过沪深港通买 A 股的外资（"聪明钱"代表） |
| **北向重仓股** | 北向持股占该股流通盘 ≥ 5%（外资重仓股，单独标记） |
| **龙虎榜** | 每日涨跌幅大的票公开的买卖席位明细。机构席位多 = 机构在介入 |
| **融资余额** | 投资者借钱买股票的总额，反映杠杆资金情绪 |

### 3. liquidity_dim（流动性 / 量价触发器）
**评估什么**：今天是不是出手时机（量价信号）

| 术语 | 含义 |
|---|---|
| **换手率** | 今日成交量 / 流通股本，反映活跃度。茅台日常 0.3%，妖股可 30%+ |
| **量比** | 今日成交量 / 过去 5 日均量。**1=平均，>2=放量，<0.7=缩量** |
| `liq_shrink_floor` | **缩量止跌**：量比<0.7 + 收阳 + 距 60 日低点≤8%（左侧抄底信号）|
| `liq_mild_breakout` | **温和放量突破**：量比 1.5~3.0 + 收阳 + 突破 20 日新高（右侧追入信号） |
| `liq_active_warmup` | **持续活跃**：5 日量比 1.2~2.0 + 当日换手率年内 60-80% 分位 |
| `liq_abnormal_vol_up` | **巨量异动**：量比>5 + 收阳 + 上影<30%（强但需警惕） |
| **RISK_distribution** | 派发风险：量比>5 + 收阴或长上影（疑似主力出货） |
| **RISK_illiquid** | 流动性差：连 5 日换手率<0.5%（难以进出） |
| **RISK_overheat** | 过热：换手>15% + 距 60 日高点≤5% + RSI>70 |

### 4. technical_dim（技术指标）
**评估什么**：技术形态有没有买点

| 术语 | 含义 |
|---|---|
| **RSI** | 相对强弱指数（0-100）。**>70=超买（过热），<30=超卖（恐慌）** |
| **MACD** | 指数平滑异同移动平均线。两根线（DIF/DEA）+ 柱状图（HIST） |
| **MACD 金叉** | DIF 上穿 DEA = 买入信号。**低位金叉**（在 0 轴下方）质量更高 |
| **MACD 零轴上穿** | DIF 从负转正 = 趋势反转信号 |
| **KDJ** | 随机指标。K/D/J 三条线，金叉买死叉卖。J 值 < 0 极超卖 |
| **BOLL（布林带）** | 上中下三轨（中=20 日均线，上下=±2σ）。突破下轨反弹是买点 |
| **BB squeeze** | 布林带收口。波动率压缩后即将放大（突破信号前奏） |
| **OBV** | 累计能量潮（成交量加权）。**OBV 创新高但价格没创=资金先于价格** |
| **顶背离** | 价格新高但 RSI/OBV 没新高（动能衰竭，看跌）|
| **底背离** | 价格新低但 RSI/OBV 没新低（恐慌底，看涨）|
| **ATR** | 平均真实波幅，衡量股票每日波动幅度。常用作动态止损（如 2×ATR）|

### 5. chips_dim（筹码）
**评估什么**：套牢盘多不多 / 浮盈盘多不多

| 术语 | 含义 |
|---|---|
| **筹码分布** | 不同价位的"持股成本"分布图（自算） |
| **chips_concentration** | 90% 筹码区间宽度 / 当前价。**越小越集中**（共识强）|
| **cost_deviation** | (当前价 − 加权平均成本) / 平均成本。**>0=多数浮盈**，<0=多数套牢 |
| **chips_peak_distance** | 当前价距筹码峰中心的距离 |

### 6. regime_dim（大盘 / 行业环境）
**评估什么**：大盘是涨还是跌、个股跑赢沪深300 没

| 术语 | 含义 |
|---|---|
| **index_regime** | 大盘环境：bull（牛）/ bear（熊）/ sideways（震荡）|
| **bull** | 上证 close > MA20 > MA60（多头排列）|
| **bear** | close < MA20 < MA60（空头排列）|
| **MA20 / MA60** | 20 日 / 60 日移动平均线 |
| **stock_rs_20d** | 个股 20 日涨跌幅 − 沪深300 同期涨跌幅（**>0 = 跑赢大盘**）|
| **沪深300** | 沪深两市市值最大的 300 只票，A 股最重要基准指数 |

## 三、决策算法

| 术语 | 含义 |
|---|---|
| **横截面 rank** | 同一天里把 N 只股票按某指标排队，转成 0~1 分位。**0=最强，1=最弱** |
| **z-score** | 标准化值 = (x − 均值) / 标准差。常用于把不同量纲指标拉到同一尺度 |
| **composite** | 综合得分 = 各维 (1−rank) × 权重 之和，归一化到 0~1。**越大越优** |
| **partial_composite** | 部分维度合成（当前 6 维，未凑齐 8 维时叫 partial）|
| **gates** | 最低门槛三道：fund_rank≤60% / flow_rank≤60% / liq_rank≤70%。任一不过则不能 BUY |
| **BUY 阈值** | composite ≥ 0.55 才能 BUY；[0.45, 0.55) → CONTINUE_WATCH；< 0.45 → DROP |
| **overheat_penalty** | 过热降权：RSI>70 时 composite 打折，RSI<25 也打折（防接飞刀）|
| **veto_hit** | 基本面硬否决命中标志（True 时 composite 强制为 0） |

## 四、回测 / 验证

| 术语 | 含义 |
|---|---|
| **anchor 日** | 回测中选取的某一历史日期作为"假装今天"的锚点 |
| **forward return** | 锚点日之后 N 天的涨跌幅（T+1, T+3, T+5）|
| **rank-IC** | 排序信息系数 = composite 排名 与 forward return 排名 的相关系数。**>0.05 即有用，>0.10 不错，>0.20 优秀** |
| **top-bot spread** | composite 前 1/3 的平均收益 − 后 1/3 的平均收益。**正值越大越好** |
| **win%** | 胜率：T+N 收益为正的占比 |
| **样本外** | 不是用来调参的数据，用来真实验证模型有没有过拟合 |
| **过拟合** | 模型在训练数据上表现很好，但换新数据就失效 |
| **walk-forward** | 每个时点用当时可获得的数据重算 signal（最严谨的回测方式）|

## 五、数据源

| 术语 | 含义 |
|---|---|
| **akshare** | Python 开源 A 股数据库，免费 |
| **新浪 stock_zh_a_daily** | 当前 K 线主源（含 outstanding_share + turnover）|
| **腾讯 stock_zh_a_hist_tx** | K 线备用源 |
| **东财 em** | 东方财富接口（部分被限流，比如 individual_info）|
| **同花顺 ths** | 主力资金流的主源（bulk 排行）|
| **百度估值** | PE/PB 时序的主源（`stock_zh_valuation_baidu`）|
| **理杏仁 lg** | 备用估值源（不稳定）|
| **bulk 接口** | 一次返回全市场所有股票（vs 单只查询，速度差 100 倍）|

## 六、其他常见术语

| 术语 | 含义 |
|---|---|
| **ST** | Special Treatment，业绩差被特殊处理的股票（有退市风险）|
| **\*ST** | 比 ST 更严重，已退市预警 |
| **A 股** | 中国大陆上市、人民币计价的普通股 |
| **科创板** | 上交所创业板，代码 688xxx，主要是科技公司 |
| **创业板** | 深交所创业板，代码 300xxx |
| **流通股本** | 市场上能自由买卖的股数（vs 总股本含限售股）|
| **流通市值** | 流通股本 × 股价 |
| **市盈率倒数** | 1/PE，衡量股票的"债券化收益率" |
| **限售解禁** | 限售股到期可以自由卖出（往往伴随股价下跌压力）|
| **龙头股** | 行业里最强的票，机构集中度高 |
| **追涨杀跌** | 涨了才追、跌了才割肉的散户行为，决策引擎想避免这种 |
